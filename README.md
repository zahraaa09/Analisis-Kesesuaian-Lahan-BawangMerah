# 🌾 SIG Web — Kesesuaian Lahan Bawang Merah

Sistem Informasi Geografis berbasis web untuk analisis kesesuaian lahan tanaman bawang merah di Kabupaten Barru.

## 🛠️ Teknologi
- **Backend**: FastAPI, PostgreSQL, PostGIS
- **Frontend**: Leaflet.js, TailwindCSS, Chart.js

## 🚀 Cara Menjalankan

### 1. Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 2. Frontend
- Buka frontend/index.html dengan Live Server

## 📊 Fitur
- Peta interaktif dengan 5 layer
- Analisis polygon
- Rekomendasi lahan
- Export CSV & Shapefile
- Dashboard chart
